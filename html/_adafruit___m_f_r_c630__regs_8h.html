Performs Mifare Classic authentication

Activates the receive circuit

MFRC630_CMD_TRANSMIT 

Transmits data from the FIFO buffer

MFRC630_CMD_TRANSCEIVE 

Transmits data from the FIFO buffer and automatically activates the receive buffer when finished

MFRC630_CMD_WRITEE2 

Gets 1 byte from FIFO and writes to EEPROM

MFRC630_CMD_WRITEE2PAGE 

Gets up to 64 bytes from FIFO and writes to EEPROM

MFRC630_CMD_READE2 

Reads data from EEPROM and copies it into the FIFO buffer

MFRC630_CMD_LOADREG 

Reads data from the internal EEPROM and initializes the MFRC630 registers. EEPROM address needs to be within EEPROM sector 2

MFRC630_CMD_LOADPROTOCOL 

Reads data from the internal EEPROM and initializes the MFRC630 registers needed for a protocol change.

MFRC630_CMD_LOADKEYE2 

Copies a key from EEPROM into the key buffer

MFRC630_CMD_STOREKEYE2 

Stores a MIFARE key (6 bytes) into EEPROM

MFRC630_CMD_READRNR 

Copies bytes from the random number generator into the FIFO buffer until the FIFO is full

MFRC630_CMD_SOFTRESET 

SW resets the MFRC630
